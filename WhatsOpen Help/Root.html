Main Getting Started Idea/Bug Reporter Root Google App App Doc Troubleshooting Credits How do I get a file listing as an administrator? If you need to you can enable administrative access in the preferences. This will cause WhatsOpen to prompt you for a username/password for administrative access when you list files. How does it work? This administrative access runs lsof as root to give you a file listing for all users instead of only the user logged in. It has potential security repercussions and is disabled by default both for convenience and practical security guidelines.
